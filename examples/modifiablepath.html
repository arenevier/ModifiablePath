OpenLayers Modifiable Path Handler Example

A demonstration of the Modifiable Path Handler to draw and modify path.

Click on the map to add points. Double click to finalize path. Once a point been added, you can modify it by selecting it then dragging it to a new location. Also, when pressing Shift key, you enter in delete mode. When in delete mode, clicking on a point of the path deletes the point. Releasing Shift key leaves delete mode.

Once a path has been finalized, you cannot it anymore.
